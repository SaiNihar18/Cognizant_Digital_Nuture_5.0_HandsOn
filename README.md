# Cognizant Digital Nurture 5.0 Hands-On Exercises

This repository contains the hands-on training exercises for Java Full Stack Engineering (FSE) Deepskilling.

---

## Directory Structure

### [Week 1]
Foundational programming concepts:
- **Design Patterns**: Builder, Adapter, Decorator, Proxy, Observer, Strategy, Command, MVC, and Dependency Injection.
- **Data Structures & Algorithms**: Search mechanisms, sorting, custom singly linked lists, and library searches.
- **PL/SQL**: Database schema, error handling, custom functions, triggers, cursors, and packages.
- **JUnit & Mockito**: Basic/advanced unit tests, parameterized testing, tagging, exception validation, and service mocking.
- **SLF4J**: Logging levels, placeholder logs, and appender configuration templates.

### [Week 2]
Enterprise application frameworks:
- **Spring Core & Maven**: Dependency injection using XML configs and annotation scanning.
- **Spring AOP**: Performance tracking logs.
- **Spring Data JPA**: Custom JPQL/native SQL finders, database auditing, and paginated searches.

### [Week 3]
Web APIs:
- **Spring REST**: CRUD controllers, validations, and global controller advice error mapping.
- **Spring Security & JWT**: Request filters and token authentication handlers.

### [Week 4]
Microservices and Gateway:
- **Eureka Server**: Service registry module.
- **Account & Loan Services**: Independent client services registering to Eureka.
- **API Gateway**: Spring Cloud Gateway routing rules and request logging filters.
